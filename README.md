<h2>需要本地配置</h2>
<p>MongoDB</p>
<p>Node.js</p>
<h2>然后通过</h2>
<p>npm install 根据package.json，自动注入依赖包</p>
<p>mongorestore characters.bson 向数据库中添加角色库 </p>
<p>mongo 启动数据库</p>
<p>glup 连接依赖文件、编译LESS样式并监视文件变化</p>
<p>npm start 启动项目</p>
